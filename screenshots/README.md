# Screenshots Directory

This directory contains all the screenshots for the CareerSU platform documentation.

## Image Files

To add images to this repository, you can:

1. **Upload via GitHub Web Interface:**
   - Go to the GitHub repository
   - Navigate to the `screenshots/` folder
   - Click "uploading an existing file"
   - Select and upload your image files

2. **Via Git Command Line:**
   ```bash
   git add screenshots/your-image.png
   git commit -m "Add screenshot: your-image.png"
   git push
   ```

## Image Guidelines

- **Format**: PNG or JPG
- **Size**: Optimize for web (under 500KB each)
- **Dimensions**: 1200px width recommended for consistency
- **Naming**: Use descriptive names matching the README references

After uploading images, they will automatically display in the README.md file.
